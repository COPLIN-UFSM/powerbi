Instalação¶

Não é necessário possuir uma licença para começar a utilizar o Microsoft Power BI.

A maneira recomendada por este curso para baixá-lo é através desta página da Microsoft, que fará download de um instalador do Windows.

O Power BI se apresenta também de outras duas maneiras: como um aplicativo do Windows, disponibilizado pela Microsoft App Store, e na versão online. Neste curso, faremos uso da versão do instalador do Windows.

Após instalar o Power BI, não é necessário fazer login ou autenticação nela para trabalhar, apesar que alguns recursos avançados (como a nuvem de compartilhamento) estarão desabilitados sem a autenticação.
