Estudantes¶

Faça download da tabela iestudantes.csv no seu computador, e abrindo-a No Power BI, faça um relatório com os seguintes itens:

  1. Número de alunos…

    • total, como um número

    • Por gênero, como um gráfico de rosca

    • Por etnia, como um gráfico de barras horizontais

    • Por modalidade (EAD ou Presencial), como um gráfico de barras verticais

    • Por nível de ensino, como um gráfico de barras horizontais

    • Por turno, como um gráfico de barras horizontais

    • Por grau acadêmico, como um gráfico de barras horizontais

    • Por centro, como um gráfico de barras horizontais

    • Um filtro para selecionar estudantes nacionais ou estrangeiros

Solução¶

Clique para expandir
  1. Carregar dados
  2. Colocar primeira linha como cabeçalho
  3. Converter colunas numéricas para números inteiros
  4. Criar medida que soma o número de alunos
  5. Criar gráficos
  6. Criar seletor
